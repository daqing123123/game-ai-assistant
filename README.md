## 🚀 快速开始

### 安装

**Godot:**
```
复制 godot/addons/game_ai_assistant/ → 项目 addons/
启用插件 → Window → Game AI Assistant
```

**Unity:**
```
复制 unity/Assets/GameAIAssistant/ → 项目 Assets/
Window → Game AI Assistant → 配置 API Key
```

### 常用命令

| 命令 | 功能 |
|------|------|
| `帮我做XXX游戏` | 生成完整游戏代码 |
| `扫描项目` | 查看项目结构 |
| `应用代码` | 保存AI生成的代码 |
| `今日学习` | 获取开发技巧 |
| `知识库` | 管理知识 |
| `添加知识:标题\|内容` | 添加知识条目 |
| `搜索知识:关键词` | 搜索知识 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `F1` | 切换助手面板 |
| `F2` | 快速搜索素材 |
| `F3` | 快速生成代码 |
| `F4` | 截取场景图 |
| `F5` | 今日学习 |
| `F6` | 打开知识库 |
